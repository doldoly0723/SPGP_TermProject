# 할로우나이트 


## 게임컨셉
할로우 나이트 모작

플랫모퍼, 액션 어드벤처, 소울라이크 

몬스터를 처치해 나아가며 최종적으로 보스를 처치하는 게임

## 원작과의 차이점
기존 원작의 메트로베니아 형식의 맵을 간단하게 일자형 형식으로 구현할 예정

## 개발 범위
#### 플레이어
이동, 점프, 공격, 스킬, 점프공격(상하좌우), 에너지 게이지

#### 몬스터
일반 몬스터1,2,3 

보스몬스터1
